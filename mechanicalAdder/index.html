Virtual Mechanical Adder

This is a binary adding machine, based on Matthias Wandel's design. While it would be more efficent in 2D, I just love the analog feel of 3D objects

Loading...

    How to use:

  1. Input the a number in binary form by clicking the pipes above the numbers.
  2. Pull the upper handle to the right to confirm the number.
  3. Repeat for as many number you want to add.
  4. Drag the lower handle to the right to get final result.

    Tips:

  • Overflowing the adder is fun!
  • You'll get some firework if you can make the ball fall out of the machine
  • Requires a browser with support for WebAssembly and WebGL 2.0. This include recent version of Chrome, Firefox, Opera.
